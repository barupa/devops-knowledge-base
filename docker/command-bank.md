# Docker Command Bank

Krátká banka nejpoužívanějších Docker příkazů.

## Basic

- Výpis lokálně dostupných image.

```bash
docker images
```

## Troubleshooting

- Pokud se image nezobrazuje, ověřte správný Docker context/daemon.

## Common Mistakes

- Záměna image (`docker images`) a kontejnerů (`docker ps`).
