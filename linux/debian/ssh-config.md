# SSH Config Examples

Praktické ukázky pro ~/.ssh/config získané z LLM odpovědí.

## Basic

<!-- Základní SSH config ukázky -->

```sshconfig
Host zkratka
HostName ip_adresa
User username
```

```sshconfig
Host debian
HostName 192.168.1.100
User martin
```

```sshconfig
Host debian
HostName 192.168.1.100
User martin
Port 2222
IdentityFile ~/.ssh/id_rsa
```
