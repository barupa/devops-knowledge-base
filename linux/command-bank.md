# Linux Command Bank

Krátká banka základních Linux příkazů.

## Basic

- Aktualizace balíčků na Debian/Ubuntu.

```bash
sudo apt update
```

## Troubleshooting

- Při chybách repozitářů zkontrolujte `sources.list` a síť.

## Common Mistakes

- `apt update` bez následného `apt upgrade`, když je cílem patching.
